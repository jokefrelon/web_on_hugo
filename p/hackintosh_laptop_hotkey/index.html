Hackintosh Laptop 常见快捷键

一般来说可以macOS和Windows对应键位参考下面的对照表即可

macOS Windows
Command / ⌘ win
Option / ⌥ alt
Control / ⌃ ctrl
Shift / ⇧ shift

所有的通用快捷键加粗标记

1. 系统快捷键:

强制退出：win + alt + esc

对标Windows的任务管理器，但是简化到了只有强制退出功能

切换桌面：ctrl + ◀︎ / ▶︎

切换任务：win + tab

打开finder：win + t

连接samba：win + k

2. 常见通用快捷键：

全选：win + a

复制字符：win + c

保存：win + s

粘贴字符：win + v

退出登陆：win + alt + shift + q

剪切字符：win + x

撤销操作：win + z

中断执行：ctrl + c

行首：win + ◀︎ / ctrl + a

行尾：win + ▶︎/ ctrl + e

页首：win + ▲

页尾：win + ▼

输入emoji：ctrl + win + space

删除选中：win + back

3. 常见软件快捷键

3.1 Typora

一级标题：win + 1

二级标题：win + 2

加粗：win + b

表格：win + t

斜体：win + i

代码：ctrl + `

下划线：win + u

超链接：win + k

无需列表：win + alt + u

有序列表：win + alt + o

显示大纲：win + shift + l / ctrl + win + 1

代码块：win + alt + c / 建议还是直接打 ``` 或者 ~~~

分割线：win + alt + -

目录生成：[toc]

3.2 Safari / Chrome

开发者模式：win + alt + i

刷新网页：win + r

切换标签页：ctrl + tab

进入/退出 全屏模式：win + ctrl + f

显示/隐藏 状态栏：win + shift + b

显示/隐藏 所有历史记录：win + y

打开收藏栏第一个：win + alt + 1

打开收藏栏第二个：win + alt + 2

在侧边栏打开收藏栏：win + ctrl + 1

在侧边栏打开稍后阅读：win + ctrl + 2

新建窗口：win + alt + n

新建无痕窗口：win + shift + n

新建标签页：win + t / win + n

关闭标签页：win + w

退出当前软件：win + q

打开软件偏好设置：win + ,

放大视图：win + +

缩小视图：win + -
